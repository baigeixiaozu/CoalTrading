# 6.13用户注册接口说明

### 6.13.1 //首次注册

接口描述：用户注册，得到基础信息。

接口地址：/userinfo/reg？

请求方法：post

填写内容示例：

{

​	”data“：{

​		“username”：“huhansan：，//用户名

​		”pwd“：“123456”，	//

​		”rpwd“：”123456“，//确认密码

​		“role”：”供应商“//身份 (供应商、需求商)

​	}

}

返回内容：

​	{

“code”：0，

“msg”：“创建成功”

}

{

“code”：1，

“msg”：“用户名重复，创建失败”

}



### 6.13.2            //获取身份信息

接口描述：二次注册界面，获得首次注册时身份信息

接口地址：/userinfo/reg2？userid=

请求方法：get

请求参数：userid //用户id

返回内容：对应id的身份供应商/采购商
