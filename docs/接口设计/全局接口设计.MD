# 全局接口设计
## 基本响应数据格式
```json
{
  "code": 0,
  "msg": "message",
  "error": "reason",
  "data": {
    
  }
}
```
|字段|释义|
|---|---|
|code|状态码|
|msg|与状态码相关的消息|
|error|与状态码相关的错误原因|
|data|响应数据，异常时为空|
## 状态码定义
>状态码由两部分组成\
> 模块码 + 模块状态码\
> 实例：100 + 201 ---> 100201\
> 其中100为模块码，201为该模块异常码

>注：全局正常状态码为 - 200\
> 全局状态码包含`通用状态码`

### 通用状态码
查阅此处：[https://www.runoob.com/http/http-status-codes.html](https://www.runoob.com/http/http-status-codes.html)
